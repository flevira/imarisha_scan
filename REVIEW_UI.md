# Review UI Workflow (Answer Sheet)

## Edit and Approval Rules
- All fields are editable: `user_id`, `question_id`, `test_id`, `exam_id`, `answer`.
- Any reviewer can approve or reject a record.
- Rejected records should be pushed to the error queue for reprocessing.

## Recommended Keyboard Shortcuts
- `Tab` / `Shift+Tab`: move across fields.
- `Up` / `Down`: move between rows.
- `A` `B` `C` `D` `E`: set answer value for focused row.
- `Ctrl+S`: save edits.
- `Ctrl+Enter`: approve current row.
- `Ctrl+Shift+Enter`: reject current row.
- `Ctrl+F`: search question ID.
- `Esc`: cancel edit for the current cell.
